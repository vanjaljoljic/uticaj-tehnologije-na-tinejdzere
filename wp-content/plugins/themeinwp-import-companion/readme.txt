== Installation ==
	= Using The WordPress Dashboard =
	* Navigate to the 'Add New' in the plugins dashboard
	* Search for ThemeinWP Import Companion
	* Click Install Now
	* Activate the plugin on the Plugin dashboard

	= Uploading in WordPress Dashboard =
	* Navigate to the 'Add New' in the plugins dashboard
	* Navigate to the 'Upload' area
	* Select themeinwp-import-companion.zip from your computer
	* Click 'Install Now'
	* Activate the plugin in the Plugin dashboard

	= Using FTP =
	* Download themeinwp-import-companion.zip
	* Extract the themeinwp-import-companion directory to your computer
	* Upload the themeinwp-import-companion directory to the /wp-content/plugins/ directory
	* Activate the plugin in the Plugin dashboard

	= Setting Options =
	* Setting Page is located as Appearance-> ThemeinWP Import Companion on your Dashboard
